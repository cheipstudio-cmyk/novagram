# Novagram

**The first custom Telegram client for Android built entirely in Kotlin + Jetpack Compose.**

A second-screen alternative to the official app, designed for people who use Telegram all day and want it to feel deliberate. Editorial dark aesthetic, serif italic titles, warm amber accents. Every animation and interaction tuned by hand — nothing left at the Material default.

Beyond the basics: AI-assisted replies, inline bot keyboards, theme sharing, chat-info media gallery, screenshot-blocked protected chats, voice notes with playback speed, 5 native languages. No calls, no Stories, no clutter.

[Get it on Google Play →](https://play.google.com/store/apps/details?id=com.secondream.novagram)
